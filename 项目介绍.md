# 智能手机人体活动识别项目 — 项目介绍

## 项目背景
本项目基于 UCI **Human Activity Recognition with Smartphones** 数据集。30 名 19–48 岁志愿者腰部佩戴智能手机（三星 Galaxy S II），通过内置**加速度计**和**陀螺仪**以 50Hz 采样三轴线性加速度与角速度，记录 6 种日常活动。信号经噪声滤波、2.56 秒滑动窗口（50% 重叠）处理后，从时域和频域提取出 **561 维特征向量**。

## 任务目标
将每条传感器记录分类为 6 种活动之一：
- **动态类**：`WALKING`（走路）、`WALKING_UPSTAIRS`（上楼）、`WALKING_DOWNSTAIRS`（下楼）
- **静态类**：`SITTING`（坐）、`STANDING`（站）、`LAYING`（躺）

## 数据概况
| 集合 | 样本数 | 特征数 | 受试者 | 缺失值 |
|------|-------|--------|--------|--------|
| 训练集 | 7352 | 561 | 21 人 | 0 |
| 测试集 | 2947 | 561 | 9 人 | 0 |

训练/测试按受试者划分（70%/30%），保证被测人员不重叠，评估更真实。

## 技术方案
1. **数据探索 (EDA)**：类别分布、受试者样本量、特征分布分析
2. **预处理**：`StandardScaler` 标准化 + `LabelEncoder` 标签编码
3. **可视化降维**：PCA / t-SNE 二维投影，直观展示类别可分性
4. **模型对比**：5 折交叉验证比较 5 种分类器
5. **最终评估**：选最优模型在独立测试集验证

## 核心结果
5 折交叉验证中 **RBF 核 SVM** 表现最佳（CV 准确率 98.75%），最终测试集：

| 指标 | 数值 |
|------|------|
| **准确率 Accuracy** | **95.42%** |
| **宏平均 F1** | **95.33%** |

- 动态类（走路/上下楼）与 `LAYING` 识别近乎完美（F1 ≥ 0.94，LAYING 达 1.00）
- 主要误差来自 **SITTING 与 STANDING 互相混淆**（两者均为静止姿态，传感器信号相近）

## 运行方式
```bash
pip install -r requirements.txt

python visualize_data.py    # 生成可视化图表
python charts_to_pdf.py     # 图表导出 PDF
```
